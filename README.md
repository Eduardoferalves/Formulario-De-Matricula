# Formulário de Matrícula

Este é um projeto de um formulário de matrícula web, desenvolvido com HTML e CSS. Ele foi projetado para ser uma interface simples e amigável para coletar informações de matrícula.

## Visão Geral

O objetivo deste projeto é fornecer um formulário de matrícula básico, mas visualmente atraente, que seja fácil de usar e integrar em outras aplicações web.

## Funcionalidades

*   **Design Responsivo:** Adaptável a diferentes tamanhos de tela (embora o responsivo não tenha sido totalmente implementado, a estrutura de pastas aponta para essa direção).
*   **Campos de Entrada Variados:** Inclui campos para texto, caixas de seleção (checkboxes) e botões de rádio (radio buttons).
*   **Estilo Moderno:** Utiliza um conjunto de estilos CSS para uma apresentação limpa e moderna.

## Estrutura do Projeto

*   `index.html`: O arquivo HTML principal que contém a estrutura do formulário.
*   `assets/`: Contém imagens e ícones usados no formulário.
    *   `Illustration.svg`: Ilustração principal.
    *   `logo.svg`: Logotipo do projeto.
    *   `icons/`: Vários ícones para campos e estados visuais.
*   `styles/`: Contém todos os arquivos CSS para estilização.
    *   `global.css`: Estilos globais e resets.
    *   `layout.css`: Estilos para o layout geral da página.
    *   `index.css`: Estilos específicos da página principal.
    *   `forms.css`: Estilos gerais para elementos de formulário.
    *   `fields/`: Estilos detalhados para tipos específicos de campos.
        *   `buttons.css`: Estilos para botões.
        *   `checkbox.css`: Estilos para checkboxes.
        *   `droparea.css`: Estilos para áreas de 'drag and drop' (se implementadas).
        *   `input.css`: Estilos para campos de texto.
        *   `radio.css`: Estilos para radio buttons.

## Como Usar

Para visualizar o formulário, basta abrir o arquivo `index.html` em seu navegador web.

## Tecnologias Utilizadas

*   **HTML5:** Estrutura do conteúdo.
*   **CSS3:** Estilização e design.
